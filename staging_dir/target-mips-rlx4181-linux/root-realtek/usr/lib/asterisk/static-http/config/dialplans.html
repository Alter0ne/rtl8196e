A Dial Plan is a collection of Outgoing Call Rules . Dial Plans are assigned to Users to specify the dialing permissions they have. For example, you might have one Dial Plan for local calling that only permits users of that Dial Plan to dial local numbers, via the "local" outgoing calling rule. Another user may be permitted to dial long distance numbers, and so would have a Dial Plan that includes both the "local" and "longdistance" outgoing calling rules.
